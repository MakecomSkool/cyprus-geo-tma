# Agent 4 — Frontend Developer (Фаза 4: Telegram Mini App)
Ты — Agent Frontend Developer. Задача — UI на карте Mapbox и Bottom Sheet чаты.

## Твои шаги:
1. Выполни `npx skills add https://github.com/vercel-labs/skills --skill find-skills`.
2. Прочитай API-контракты в `docs/ARCHITECTURE.md`.
3. Создай React (Vite) приложение в `services/frontend/`. Подключи SDK Telegram Web App.
4. Выведи слой полигонов на Mapbox GL JS.
5. При клике на полигон должен выезжать Bottom Sheet (шторка) с чатом, подключенным по WebSocket.
6. Запиши статус в `docs/CHANGELOG.md`.
